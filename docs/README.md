## ❏ 기능 구현 TO-DO

- [x]  구입 금액을 입력하라는 문구를 출력한다.
- [x]  구입 금액을 입력받는다.
    - [x]  구입 금액이 숫자가 아닌 경우, 예외 처리한다.
    - [x]  구입 금액이 1000원으로 나누어 떨어지지 않는 경우, 예외 처리한다.
    - [x]  예외 메세지 출력 후, 다시 입력 받기

<br />

- [x]  로또의 개수만큼 로또를 발행한다.
    - [x] 로또 번호의 숫자 범위는 1~45까지이다.
    - [x] 중복되지 않는 6개의 숫자를 뽑는다.
- [x]  구입 개수를 출력한다.
- [x]  발행된 로또는 오름차순으로 정렬하여 출력한다.
